# Session Protocol

## On Session Start
1. Check `shared/memory-sessions.md` for where we left off
2. Check `shared/memory-decisions.md` for pending decisions
3. Verify current git status and branch

## During Session
- Update memory files AS YOU GO when learning something new
- Mark completed tasks immediately in todo list
- Test changes with `npm run build` + `npm run lint` before suggesting deploy

## On Session End
- Update `shared/memory-sessions.md` with work completed
- Note any pending items or blockers
- If decisions were made, update `shared/memory-decisions.md`
